## Constraints

- No new native dependencies (PNG decode stays pure-JS, Puppeteer reused).
- Don't break the artifact pipeline: every artifact script must still produce byte-identical output for the same inputs.
- Don't leak workspace paths into version control.
- The provenance metadata file is the only thing the runtime is allowed to trust as "approved".

## Non-goals

- Generating the actual production PNGs in this task. Image generation is a manual imagegen step driven by `npm run game:season:next`.
- Replacing the existing emoji glyph badges visually — that happens *after* every PNG ships and gets approved provenance.
